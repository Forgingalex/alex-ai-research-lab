# Research Log

This directory contains daily and weekly research logs documenting discoveries, insights, and progress on the AI Brain project.

## Purpose

Research logs help track:
- Daily discoveries and insights
- Ideas explored
- Questions raised
- Research activities
- Connections between concepts
- Next steps and priorities

## Template

See [templates/research-log-template.md](../templates/research-log-template.md) for a template.

## Naming Convention

Use dates in filenames: `YYYY-MM-DD-research-log.md` or `YYYY-MM-DD-weekly-log.md`
