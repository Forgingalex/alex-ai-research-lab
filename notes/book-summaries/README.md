# Book Summaries

This directory contains summaries of books relevant to the AI Brain research project.

## Purpose

Book summaries capture:
- Key concepts and ideas
- Relevant insights for the AI Brain
- Architecture implications
- Research directions inspired by the book

## Format

While there's no strict template, include:
- Book title and author
- Key takeaways
- Relevance to AI Brain
- Concepts to explore further

## Naming Convention

Use format: `[author-lastname]-[year]-[book-title].md`

Example: `russell-2021-human-compatible.md`
