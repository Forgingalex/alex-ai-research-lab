# Research Notes

This directory contains technical research notes, concept explanations, and detailed documentation on various topics related to the AI Brain architecture.

## Purpose

Research notes document:
- Technical concepts and theories
- Algorithm explanations
- Architecture implications
- Module designs
- Research questions
- Experiment proposals

## Template

See [docs/templates/research-note-template.md](../docs/templates/research-note-template.md) for a template.

## Naming Convention

Use descriptive, lowercase names with hyphens: `episodic-memory-design.md`, `reasoning-engine-architecture.md`
