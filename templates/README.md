# Templates

This directory contains templates for creating consistent, well-structured research documents.

## Available Templates

- **[Research Log Template](research-log-template.md)** — For daily/weekly research logs
- **[Research Note Template](research-note-template.md)** — For technical research notes
- **[Paper Summary Template](paper-summary-template.md)** — For paper reviews and summaries
- **[Experiment Template](experiment-template.md)** — For experiment documentation
- **[Architecture Proposal Template](architecture-proposal-template.md)** — For architecture designs
- **[Idea Template](idea-template.md)** — For capturing ideas

## Usage

1. Copy the relevant template
2. Fill in the sections
3. Save to the appropriate directory
4. Follow naming conventions (see `docs/naming-conventions.md`)

## Guidelines

- Use templates as starting points, but adapt as needed
- Maintain consistency in formatting
- Link related documents
- Include diagrams (Mermaid) when helpful
- Reference related research

See [Research Workflow](../docs/research-workflow.md) for more details.
